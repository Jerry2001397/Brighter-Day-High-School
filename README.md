# Brighter Day ELEM, Jr & Sr High School Website

Welcome to the official website repository for Brighter Day Elementary, Junior & Senior High School.

## About

This website provides information about our school, including:
- School mission and vision
- Educational programs for all levels (Elementary, Junior High, Senior High)
- Contact information
- Enrollment details

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional interface with smooth animations
- **Easy Navigation**: Single-page design with smooth scrolling
- **Contact Form**: Easy way for parents and students to get in touch

## Getting Started

To view the website locally:

1. Clone this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required!

## Website Structure

- `index.html` - Main website structure
- `styles.css` - Styling and responsive design
- `script.js` - Interactive features and animations

## Sections

1. **Home** - Hero section with welcome message
2. **About** - School mission, vision, and values
3. **Programs** - Detailed information about Elementary, Junior High, and Senior High programs
4. **Contact** - Contact form and school information

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
- Responsive Design principles
