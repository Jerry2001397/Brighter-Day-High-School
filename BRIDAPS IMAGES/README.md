# Image Placeholder Guide

This directory should contain the following images:

## Required Images

1. **logo.png** - School logo (BRIDAPS logo)
   - Save the school logo image here
   - Recommended size: 200x200 pixels or larger
   - Format: PNG with transparent background preferred

2. **admin-placeholder.jpg** - Administrator photos
   - Professional headshots of administrators
   - Recommended size: 400x500 pixels
   - Format: JPG

3. **teacher-placeholder.jpg** - Teacher photos
   - Professional headshots of teaching staff
   - Recommended size: 400x500 pixels
   - Format: JPG

4. **graduate-placeholder.jpg** - Graduate photos
   - Photos of graduates
   - Recommended size: 400x500 pixels
   - Format: JPG

5. **alumni-placeholder.jpg** - Alumni spotlight photos
   - Photos of featured alumni
   - Recommended size: 300x300 pixels
   - Format: JPG

## How to Add Images

1. Copy your images to this `images` folder
2. Name them according to the list above
3. Update the HTML files if you want to use different filenames
4. Ensure images are optimized for web (compressed but good quality)

## Image Optimization Tips

- Use tools like TinyPNG or ImageOptim to compress images
- Keep file sizes under 500KB for faster loading
- Use JPG for photos, PNG for logos with transparency
- Resize images to appropriate dimensions before uploading

## Note

The placeholder images referenced in the HTML files should be replaced with actual photos of your school's administrators, teachers, and students.
